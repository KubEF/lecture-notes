\chapter{Жорданова нормальная форма}

$\mca \in \End V$, $\chi_{\mca}$ раскладывается на линейные множители.

$p| \chi_{\mca}$ неприведенная приведенная $\Rightarrow p = X - \lambda$, $\lambda$ - собственное значение $\mca$.
$W_{X-\lambda} = R_{\lambda} = \{v | \exists j:(X-\lambda)^j (\mca) (v) = 0\} =
    \{v | (\mca - \lambda \epsilon)^j (v) = 0}$ - корневые векторы, принадлежащее собственному значению $\lambda$. $R_{\lambda}$ - корневое подпространство, принадлежащее собственному значению $\lambda$.

\begin{propertylist}
    \begin{enumerate}
        \item $V_{\lambda} \in R_{\item}, \ V_{\lambda} = \Ker(\mca - \lambda\epsilon)$
        \item $R_{\lambda} \neq 0 \Leftrightarrow \lambda$  - собственное значение $\mca$.
    \end{enumerate}
\end{propertylist}
